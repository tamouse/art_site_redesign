In a fit of inspiration after watching and reading some more design instruction, and just wanting something better, I decided to start re-designing this art blog from the ground up.

Originally I had used Twitter Bootstrap to put the site together, and that was cool, but this time I'm building it all out of my own things.

I'm following the [Inverted Triangle CSS]() design principles that I'd taught in the [Styling with Sass]() [GDI Minneapolis]() class. I figured, too, that if I'm teaching these things, I darn well aught to be using them as well!

The site is still statically compiled using Jekyll, but the new Jekyll Theming elements starting in the 3.3 release make doing stuff a bit easier. The design I'm building for this site I hope will be useful across my other blog/news/info sites on `tamouse.org`. If I can build it out here, I'll be able to turn it into a theme I share with the other sites, something I've been hoping to be able to do for a while.

For this re-design, I'm definitely going mobile-first. I even drew the pages out on paper first in their mobile format:

I'm using some different fonts as well:

  • Eczar is the base font.
  • Work Sans is the heading and button font.
  • Space Mono is the monospaced font.
